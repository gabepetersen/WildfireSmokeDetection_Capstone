About Team #24's Project

The project we have decided to pursue is a Wildfire Smoke Detection application. This project is intended to be used by first responders and homeowners in rural areas of California. We have a dataset of over 900 video files for which we will use to train a neural network to identify the presence of smoke and offer a prediction on its density as well. Ideally, we want to make this network minimize false positives and maximize true negatives. We plan on combining many different topics of Computer Science during the development of this project. This includes Machine Learning, Databases, Application Development, and Big Data. Furthermore, we will develop an application to allow a user to record and submit their own video to our trained model and receive the models feedback in real-time. For this project, we have spent time researching smoke detection, performing stakeholder interviews, and defining requirements for the project. During our research, we discovered a few similar projects to our idea where they detect the presence of smoke, each pursuing a different approach. Also coming out of the research stage of our project, we discovered a few websites discussing how to use the tools that we plan on utilizing, PySpark and TensorFlow.
